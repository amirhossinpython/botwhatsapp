ویژگی‌ها:
رابط کاربری گرافیکی (GUI):
با استفاده از کتابخانه Tkinter طراحی شده که به کاربران امکان می‌دهد سوالات خود را به سادگی وارد کرده و پاسخ‌ها را مشاهده کنند.

ارتباط با API:
درخواست کاربر به یک وب‌سرویس ارسال می‌شود و پاسخ پردازش‌شده توسط API دریافت می‌شود.

ذخیره و مدیریت پاسخ:
پاسخ‌های دریافتی در یک فایل متنی ذخیره شده و برای سهولت به کلیپ‌بورد کپی می‌شود.

ارسال پیام به واتساپ:
پیام‌ها با استفاده از کتابخانه PyAutoGUI به صورت خودکار به واتساپ ارسال می‌شوند.

کاربرد آسان:
بدون نیاز به دانش فنی خاص، کاربران می‌توانند برنامه را اجرا کرده و از امکانات آن استفاده کنند.

پیش‌نیازها:
برای اجرای این پروژه، ابتدا باید پیش‌نیازهای زیر را نصب کنید:

نصب Python:
اطمینان حاصل کنید که Python  روی سیستم شما نصب باشد.

نصب کتابخانه‌های مورد نیاز:
با استفاده از دستور زیر، کتابخانه‌های مورد نیاز را نصب کنید:
pip install pyautogui requests pyperclip

واتساپ وب باز باشد:
قبل از اجرای برنامه، باید واتساپ وب در مرورگر شما باز و آماده ارسال پیام باشد.

نحوه اجرا:
فایل کد (botwhatsapp.py) را دانلود کرده و ذخیره کنید.
فایل را اجرا کنید:
python botwhatsapp.py
پس از اجرا، رابط کاربری برنامه نمایش داده می‌شود.
متن یا سوال خود را در کادر ورودی وارد کنید و دکمه "دریافت و ارسال" را فشار دهید.
برنامه متن شما را به API ارسال کرده، پاسخ را دریافت می‌کند، و پیام را در واتساپ ارسال می‌کند.

نحوه کار برنامه:
ورودی کاربر:
کاربر سوال یا متن خود را وارد کرده و با فشار دادن دکمه، برنامه شروع به کار می‌کند.

ارتباط با API:
متن واردشده به API ارسال می‌شود و پاسخ پردازش‌شده دریافت می‌گردد.

ذخیره و کپی:
پاسخ دریافت‌شده در یک فایل متنی ذخیره شده و به صورت خودکار به کلیپ‌بورد کپی می‌شود.

ارسال به واتساپ:
برنامه با استفاده از PyAutoGUI پیام را در واتساپ وارد کرده و ارسال می‌کند.

نکات مهم:
API مورد استفاده باید قابل دسترسی باشد؛ در غیر این صورت ممکن است خطا دریافت کنید.
واتساپ وب باید قبلاً در مرورگر شما باز شده باشد تا برنامه بتواند پیام را ارسال کند.
حتماً از پایتون نسخه 3.6 به بالا استفاده کنید.
